# 01_What_Is_FPGA — 什么是 FPGA？

## 核心概念

**FPGA (Field Programmable Gate Array)** — 现场可编程门阵列

FPGA 是一种半导体器件，你可以在制造出来**之后**通过编程来配置它的硬件电路。这与 ASIC（专用集成电路）不同——ASIC 一旦制造就无法更改。

## FPGA 内部结构

```
┌─────────────────────────────────────────────┐
│                  FPGA Chip                   │

│  │ CLB │─│ CLB │─│ CLB │─│ CLB │  ...     │
│  └──┬──┘ └──┬──┘ └──┬──┘ └──┬──┘          │
│     │       │       │       │              │
│  ═══╪═══════╪═══════╪═══════╪═══  Routing   │
│     │       │       │       │              │
│  ┌──┴──┐ ┌──┴──┐ ┌──┴──┐ ┌──┴──┐          │
│  │ CLB │─│ CLB │─│ CLB │─│ CLB │  ...     │
│  └─────┘ └─────┘ └─────┘ └─────┘          │
│                                              │
│  ┌─────┐  ┌─────┐  ┌─────┐                 │

│  └─────┘  └─────┘  └─────┘                 │

```

### CLB (Configurable Logic Block) — 可配置逻辑块
每个 CLB 包含：
- **LUT (Look-Up Table)**：查表，实现任意组合逻辑。一个 N 输入 LUT 可以实现任意 N 输入的布尔函数
- **Flip-Flop (FF)**：触发器，实现时序逻辑（寄存器）
- **MUX**：多路选择器，选择数据通路

### 其他资源
- **IOB (I/O Block)**：输入输出单元，连接外部引脚
- **BRAM**：块 RAM，片上存储器
- **DSP Slice**：专用乘加单元
- **PLL/MMCM**：时钟管理单元

## FPGA vs CPU vs ASIC

| 特性 | CPU | FPGA | ASIC |
|------|-----|------|------|
| 执行方式 | 取指-执行（软件） | 硬件并行 | 固定硬件 |
| 灵活性 | 高（软件可改） | 高（比特流可改） | 低（流片后不可改） |
| 性能 | 中等 | 高（并行） | 最高 |
| 功耗 | 中等 | 中低 | 最低 |
| 开发周期 | 短 | 中等 | 长（数月） |
| NRE成本 | 低 | 中 | 极高（百万级） |
| 单价 | 几~几十美元 | 几~几百美元 | 批量后极低 |

### 一句话总结
> CPU 通过软件实现灵活性，ASIC 通过定制实现极致性能，FPGA 在两者之间提供了可重配置的硬件并行能力。

## LUT 工作原理（以 2 输入 LUT 为例）

一个 2 输入 LUT 本质上是一个 4x1 的 SRAM：

```
输入: A, B
输出: Y

SRAM 内容（真值表）:
地址(A,B=00): Y = ?
地址(A,B=01): Y = ?
地址(A,B=10): Y = ?
地址(A,B=11): Y = ?

通过修改 SRAM 内容，2输入 LUT 可以实现 16 种不同的逻辑函数:

AND:  内容 = 0001 → Y = A & B
OR:   内容 = 0111 → Y = A | B
XOR:  内容 = 0110 → Y = A ^ B
NOT:  内容 = 0011 → Y = ~A (B 不用)
...
```

Xilinx 7 系列 FPGA 使用 6 输入 LUT（LUT6），可存储 64 位，实现任意 6 输入布尔函数。

## Demo: LED 闪烁（Hello World of FPGA）

参见 `01_led_blink.v` — 这是最简单的 FPGA 设计，实现 LED 每 0.5 秒闪烁一次。

### 设计思路
1. 系统时钟（如 50MHz）驱动计数器
2. 计数器到达阈值时翻转 LED 状态
3. 50MHz = 50,000,000 次计数/秒，0.5 秒 = 25,000,000 次计数
