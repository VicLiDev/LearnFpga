# 01_Introduction — 入门篇

本章节面向零基础初学者。学习目标：
- 了解什么是 FPGA，它能做什么
- 搭建完整的开发环境
- 学习 Verilog HDL 基本语法
- 完成第一个 FPGA 项目（LED 闪烁）

- 基本数字逻辑概念（0/1、与或非门）
- 能够在电脑上安装软件
- 耐心 💪
