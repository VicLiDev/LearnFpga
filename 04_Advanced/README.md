## FPGA 做 DSP 的优势

```
传统 DSP 芯片                    FPGA
  单指令流执行                    大规模并行执行
  串行处理                        自定义数据通路
  固定计算单元                    灵活的拓扑结构
  受限于时钟频率                  突破时钟频率瓶颈

应用场景: 雷达、通信、音频处理、电机控制
```
